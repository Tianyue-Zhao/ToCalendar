Convert to icalendar!

How to use

This page converts homework calendars in tables in word documents into icalendar files ready for importing into calendar applications such as Google Calendar, Outlook, Apple Calendar, and Schoology. Students will never need to check the homework calendar document again! Select the docx file containing the word document using the form below, enter the periods you need calendars for, click the generate button, and you will get a zip file containing the importable icalendar files!

Step 2

Please enter the periods you need the calendars for in the following box.
Use single letters not separated by anything, as in "BCF" to generate calendars for periods B,C and F.

Step 3

When you're done, click Convert and save the zip file that contains the homework calendars!
