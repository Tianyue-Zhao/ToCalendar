Convert to icalendar!

About

This page converts homework calendars in tables in word documents into icalendar files ready for importing into calendar applications such as Google Calendar, Outlook, Apple Calendar, and Schoology. Students will never need to check the homework calendar document again!

Video tutorial for using this website

Step 1

Step 2

Please enter the periods you need the calendars for in the following box.
Use single letters not separated by anything, as in "BCF" to generate calendars for periods B,C and F.

Step 3

Please enter the name of the course your calendar is for.

Step 4

When you're done, click Convert and save the zip file that contains the homework calendars!
