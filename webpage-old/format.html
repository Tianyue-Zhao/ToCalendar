Tips for using ToiCalendar


First of all...

ToiCalendar scans through the document row by row, so each row of your homework document should contain an assignment, while the first row contains headers, like this:


Date

Each assignment should have a date, which can be written in two ways:

Start of week and day of week

The popular "start of week + day of week" representation works with ToiCalendar, as long as:
The "start of week" column is labeled "week" in the header And... The "day of week" column is labeled "day" in the header

Month and day

If you prefer using the month and day representation, that's find as well! Just label the column as "date" in the header:
